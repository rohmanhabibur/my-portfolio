# Professional Web Portfolio - Habibur Rohman, S.Kom

Repositoli Ini Berisi Kode Sumber Situs Web Portofolio Profesional Yang Dirancang Khusus Untuk Menampilkan Keahlian Di Bidang **Full-Stack Web Development** Serta **Sistem Otomatisasi Administrasi Data**. Proyek Ini Dideploy Menggunakan Layanan **Cloudflare Pages** Untuk Performa Akses Yang Maksimal, Cepat, Dan Aman.

## 🚀 Fitur Utama Website

*   **Desain Modern & Responsif:** Menggunakan Tailwind CSS Untuk Pengalaman Pengguna Yang Optimal Di Semua Perangkat (Mobile, Tablet, Desktop).
*   **Segmentasi Keahlian Khusus:** Menonjolkan Kompetensi Utama Di Bidang PHP Frameworks Dan Integrasi Otomatisasi Cloud.
*   **Struktur Kode Bersih:** Ditulis Dengan Mengikuti Kaidah HTML5 Semantik Yang Ramah SEO Dan Mudah Dimodifikasi.
*   **Optimasi Cloudflare:** Konfigurasi Statis Penuh Tanpa Beban Server Tambahan (Zero Server Overheads).

## 🛠️ Spesifikasi Teknologi & Keahlian Yang Ditampilkan

*   **Core Languages:** PHP, HTML5, CSS3, JavaScript (ES6+).
*   **Frameworks:** Laravel, CodeIgniter, Tailwind CSS.
*   **Cloud & Automation Integration:** Google Apps Script (GAS), Google Sheets API, Excel VBA Integration.
*   **Databases:** MySQL, PostgreSQL.
*   **Infrastructure Hosting:** Cloudflare Pages (Continuous Integration Dari Branch `main`).

## 📁 Struktur Berkas Repositori

```text
├── index.html      # Berkas Utama Halaman Tunggal Portofolio (HTML5 & Tailwind CSS)
└── README.md       # Dokumentasi Teknis Dan Panduan Repositori Proyek
```

## ⚙️ Panduan Menjalankan Secara Lokal

Untuk Membuka Dan Menguji Website Ini Di Komputer Lokal Anda, Ikuti Prosedur Singkat Berikut:

1.  **Kloning Repositori:**
    ```bash
    git clone https://github.com/rohmanhabibur/my-portfolio.git
    ```
2.  **Masuk Ke Direktori:**
    ```bash
    cd my-portfolio
    ```
3.  **Jalankan Berkas:**
    Buka Berkas `index.html` Secara Langsung Menggunakan Peramban Pilihan Anda (*Double Click* Pada Berkas Atau Gunakan Ekstensi *Live Server* Di VS Code).

## 🔄 Pembaruan Otomatis (CI/CD)

Proyek Ini Terhubung Langsung Ke **Cloudflare Pages**. Setiap Kali Perubahan Kode Dilakukan Melalui *Commit* Dan Ditekan (*Push*) Ke *Branch* `main` GitHub, Cloudflare Pages Akan Secara Otomatis Memicu Proses *Build* Dan Melakukan *Deployment* Versi Terbaru Dalam Hitungan Detik.
